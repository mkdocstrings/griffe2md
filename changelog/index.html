Changelog¤

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and this project adheres to Semantic Versioning.

1.4.0 - 2026-03-06¤

Compare with 1.3.4

Features¤

  • Improve spacing in templates directly (58e7126 by Timothée Mazzucotelli). Issue-19

Code Refactoring¤

  • Don't format code with mdformat by default (6b8928f by Timothée Mazzucotelli). Issue-19
  • Mark configuration typed dict as non-total (3636cab by Timothée Mazzucotelli).

1.3.4 - 2026-02-20¤

Compare with 1.3.3

Build¤

  • Depend on griffelib instead of griffe (5c9b7e0 by Timothée Mazzucotelli).

1.3.3 - 2026-02-05¤

Compare with 1.3.2

Bug Fixes¤

  • Remove extra }} in Jinja template (cb8aeaf by Timothée Mazzucotelli). Issue-24
  • Escape pipes in table cells (0e1da47 by Timothée Mazzucotelli). Issue-22

1.3.2 - 2025-11-20¤

Compare with 1.3.1

Bug Fixes¤

  • Revert passing options to mdformat (ab1561d by Timothée Mazzucotelli).

1.3.1 - 2025-11-20¤

Compare with 1.3.0

Bug Fixes¤

  • Don't try using the mdformat tables extension (859c3c6 by Timothée Mazzucotelli).

1.3.0 - 2025-11-20¤

Compare with 1.2.6

Features¤

  • Add parameter to prevent formatting Markdown with mdformat (8cfd626 by Timothée Mazzucotelli). Issue-18

1.2.6 - 2025-11-10¤

Compare with 1.2.5

Bug Fixes¤

  • Replace newlines with <br> tags in table descriptions (afd7ef0 by Timothée Mazzucotelli). Issue-16

1.2.5 - 2025-10-02¤

Compare with 1.2.4

Bug Fixes¤

  • Mark admonition as containing Markdown, add newlines for readability/parsers (08d8051 by Timothée Mazzucotelli). Issue=15

1.2.4 - 2025-09-24¤

Compare with 1.2.3

Build¤

  • Depend on Griffe 1.14 (ba2595f by Timothée Mazzucotelli).

Bug Fixes¤

  • Support Griffe loader options, correctly pass them to the loader (8b4363b by Timothée Mazzucotelli). Issue-13
  • Fix table rendering issue (no newline after separation or between items) (cfe0f3d by Timothée Mazzucotelli).

1.2.3 - 2025-09-09¤

Compare with 1.2.2

Bug Fixes¤

  • De-duplicate summary sections when they already exist in the docstring (6133b2a by Timothée Mazzucotelli). Issue-12

1.2.2 - 2025-08-24¤

Compare with 1.2.1

Bug Fixes¤

  • Always wrap headings in backticks (e9de94f by Timothée Mazzucotelli). Issue-11
  • Remove extra new lines in attribute headings, and don't render attribute values in headings (f6d962d by Timothée Mazzucotelli). Issue-11

1.2.1 - 2025-08-19¤

Compare with 1.2.0

Build¤

  • Update entrypoint to avoid deprecation warning (850ec71 by Timothée Mazzucotelli).

1.2.0 - 2025-08-19¤

Compare with 1.1.0

Features¤

  • Read config from config file or pyproject.toml (c29e483 by Rafal Krupinski). Issue-8, PR-10

Code Refactoring¤

  • Move modules under internal folder, re-expose API from top-level (151a6ef by Timothée Mazzucotelli).

1.1.0 - 2025-02-11¤

Compare with 1.0.2

Build¤

  • Drop support for Python 3.8 (823936b by Timothée Mazzucotelli).

Features¤

  • Allow passing partial configuration to relevant functions (4f7682e by Timothée Mazzucotelli).

1.0.2 - 2024-09-27¤

Compare with 1.0.1

Bug Fixes¤

  • Fix spacing around = and -> (e62d67c by Timothée Mazzucotelli). Issue-1

1.0.1 - 2024-08-15¤

Compare with 1.0.0

Build¤

  • Depend on Griffe 0.49 (e7fef87 by Timothée Mazzucotelli).

Code Refactoring¤

  • Update code for Griffe 0.49 (b321b39 by Timothée Mazzucotelli).

1.0.0 - 2024-01-03¤

Compare with first commit

Dependencies¤

  • Bump required Griffe version to 0.36 (7fd172e by Timothée Mazzucotelli).

Features¤

  • Resolve external aliases pointing to objects in corresponding private packages (ast.AST -> _ast.AST) (5630fdf by Timothée Mazzucotelli).
  • Support latest Griffe, add auto-summaries and cross-references (e744fac by Timothée Mazzucotelli).
  • Add initial rendering feature (d6e23ad by Timothée Mazzucotelli).

Bug Fixes¤

  • Use all members as well when not grouping by category (b7086c7 by Timothée Mazzucotelli).
  • Never fail when trying to format code with Black (940aa7a by Timothée Mazzucotelli).
  • Add back logger to rendering module (29ff6f8 by Timothée Mazzucotelli).
  • Don't import stuff from mkdocstrings (1108a3f by Timothée Mazzucotelli).
  • Add missing jinja2 dependency (4b9067b by Timothée Mazzucotelli).

Code Refactoring¤

  • Never use full path in separate signatures (ddfeac4 by Timothée Mazzucotelli).
  • Use html_links option, stop using Textual click links (f916d90 by Timothée Mazzucotelli).
